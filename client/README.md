# 💻 Budget Tracker Client

A modern, high-performance UI for the Budget Tracking App.

## ✨ Features

- **Next.js 15 App Router**: Leveraging latest React features.
- **Type Safety**: Fully typed data fetching and props.
- **Secure Auth**: Integration with HttpOnly cookie system.

## 🛠 Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS

## 🚀 Getting Started

1. `npm install`
2. `npm run dev`

## 🔒 Security Note

Authentication is handled via backend's `HttpOnly` cookies, resistant to XSS-based token theft.
